# Perf

http://stackoverflow.com/questions/35032182/decrement-index-in-a-loop-after-swift-c-style-loops-deprecated

```swift
var i = 0
for index in stride(from: 10, to: 0, by: -1) {
    i += 1
}
```
33 times slower than:
```swift
var i = 0
var j = 10
while j > 0 {
    i += 1
    j -= 1
}
```
